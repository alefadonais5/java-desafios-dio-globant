# Java - Desafios DIO | Globant Java & Spring Boot AI Developer 🚀

Repositório criado para registrar minha evolução prática no bootcamp **Java & Spring Boot AI Developer**, da DIO em parceria com a Globant.

Os desafios desenvolvidos têm foco em fundamentos da linguagem Java, lógica de programação e estruturas de controle.

## Desafios desenvolvidos

### 1. Validação de Cadastro: Nome e Idade
Objetivo:
Criar uma validação para aprovação de cadastros com base em regras de negócio.

Conceitos praticados:
- Estruturas condicionais
- Operadores lógicos
- Manipulação de Strings
- Entrada de dados

---

### 2. Contagem de Códigos de Tarefa
Objetivo:
Percorrer uma lista e identificar quantas vezes um determinado código aparece.

Conceitos praticados:
- Arrays
- Laços de repetição
- Comparação de Strings
- Controle de fluxo

---

### 3. Validação de Comandos Críticos
Objetivo:
Validar comandos permitidos e rejeitar entradas inválidas.

Conceitos praticados:
- Operadores lógicos
- Estruturas condicionais
- Comparação de Strings
- Regras de validação

## Tecnologias utilizadas

- Java
- Lógica de Programação
- Estruturas de Controle
- Orientação a Objetos (em andamento)

## Objetivo

Construir uma base sólida em Java e desenvolvimento back-end, praticando resolução de problemas e fortalecendo habilidades para projetos reais.
